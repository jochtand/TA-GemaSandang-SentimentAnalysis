# TA-GemaSandang-SentimentAnalysis

Repository penelitian Tugas Akhir:

**Implementasi Analisis Sentimen untuk Mendukung Sistem Rekomendasi Produk Fashion pada Platform Gema Sandang**

## Repository Contents

- dataset/
  - Dataset sintetis dan generator dataset
- notebook/
  - Notebook pelatihan model dan evaluasi statistik
- flask_api/
  - Implementasi Flask API beserta model Support Vector Machine
- requirements.txt
  - Daftar dependency Python

## Final Implementation

Model implementasi yang digunakan pada sistem adalah Support Vector Machine (SVM) dengan TF-IDF sebagai representasi fitur.
